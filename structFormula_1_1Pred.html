void Formula::Pred::applyToAsTerm ( std::function< void(Term *)>  termContext) const

Interpret a predicate as a term and apply an arbitrary function to it.

Provides a memory safe context in which the predicate is interpreted as a term. This is useful as it allows us to take advantage of the shared structure underlying predicates and terms
